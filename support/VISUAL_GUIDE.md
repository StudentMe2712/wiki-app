# 🎨 Визуальный гид интерфейса Wiki

## Обзор интерфейса

Wiki имеет классический трехуровневый макет:

```
┌─────────────────────────────────────────────────────┐
│ 🔵 ШАПКА (Header)                                   │
│ InEx Учет онлайн | Вошли как a.Popov | Выйти       │
├─────────────────────────────────────────────────────┤
│ 🔘 ПАНЕЛЬ НАВИГАЦИИ                                 │
│ [+Новая wiki-страница] Wiki Файлы Настройки...      │
├──────────────┬──────────────────────────────────────┤
│              │                                      │
│ 📋 МЕНЮ      │  📄 КОНТЕНТ                         │
│ СЛЕВА        │  Главная страница                    │
│              │  или                                 │
│ • Org 1      │  Страница организации                │
│ • Org 2      │  с кнопками редактирования           │
│ • Org 3      │                                      │
│              │                                      │
└──────────────┴──────────────────────────────────────┘
```

---

## 1. Шапка сайта (Header)

**Цвет**: Синий градиент (#628db6 → #1c4a6d)  
**Высота**: ~40px

```
┌────────────────────────────────────────────────────┐
│ 📦 InEx │ Учет онлайн    Вошли как a.Popov │ Выйти │
└────────────────────────────────────────────────────┘
```

**Элементы**:
- Логотип "InEx" (желтый квадрат с черным текстом)
- Название "Учет онлайн" (белый текст)
- Информация о пользователе (справа)
- Ссылки "Моя учётная запись" и "Выйти"

**Примечание**: Пользовательская информация статична (без реальной авторизации).

---

## 2. Панель навигации (Navbar)

**Цвет**: Светло-синий (#759fcf)  
**Высота**: ~36px

```
┌────────────────────────────────────────────────────┐
│ [➕ Новая wiki-страница] Обзор Действия Wiki ...  │
└────────────────────────────────────────────────────┘
```

**Элементы**:
- **Зеленая кнопка** "+ Новая wiki-страница" (слева)
- Вкладки: Обзор, Действия, Задачи, Трудозатраты, Диаграмма Ганта, Календарь, Документы
- **Wiki** (активная вкладка - темно-синий фон)
- Файлы, Настройки

**Функциональность**:
- Клик по "+ Новая wiki-страница" → открывается модальное окно создания

---

**Цвет фона**: Белый  
**Граница**: Серая справа

```

│   Wiki       │ ← Заголовок
├──────────────┤
│ • ALPHUTA    │

│ • Alanda     │
│ • BIRLIK...  │
│ • CityTrade  │
│ ...          │
└──────────────┘
```

**Элементы**:
- Заголовок "Wiki" (серый фон, 16px)
- Список организаций:
  - Каждая строка - ссылка
  - Текст синий (#0066cc)
  - Hover: светло-серый фон
  - Активная: голубой фон (#e3f2fd)

**Прокрутка**: Если организаций много - появляется скроллбар

## 5. Страница организации (View Mode)

Отображается при выборе организации из меню

```
┌──────────────────────────────────────────────────┐
│ ALPHUTA                     [✏️ Редактировать]   │
│                             [🗑️ Удалить]        │
│                             [⬇️ Экспортировать]  │
├──────────────────────────────────────────────────┤
│ ┌──────────────────────────────────────────────┐ │
│ │                                              │ │
│ │  Содержимое организации                      │ │
│ │  (Текстовое поле с предварительным просм.)   │ │
│ │                                              │ │
│ │  kuadmin123 - Пароль от Ани деск            │ │
│ │  Ku@dmin123Q - mstsc                        │ │
│ │  ...                                         │ │
│ │                                              │ │
│ └──────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────┘
```

**Заголовок**:
- Название организации (24px, жирный)
- Кнопки справа:
  - "✏️ Редактировать" (синяя)
  - "🗑️ Удалить" (серая)
  - "⬇️ Экспортировать" (серая с выпадающим меню)

**Контент**:
- Серый фон (#f9f9f9)
- Рамка (#ddd)
- Моноширинный шрифт (Courier New)
- Текст "как есть" (без форматирования Markdown)

## 6. Страница организации (Edit Mode)

Активируется кнопкой "✏️ Редактировать"

```
┌──────────────────────────────────────────────────┐
│ Название организации:                            │
│ ┌──────────────────────────────────────────────┐ │
│ │ ALPHUTA                                      │ │
│ └──────────────────────────────────────────────┘ │
│                                                  │
│ Содержимое:                                      │
│ ┌──────────────────────────────────────────────┐ │
│ │ kuadmin123 - Пароль от Ани деск             │ │
│ │ Ku@dmin123Q - mstsc                         │ │
│ │                                              │ │
│ │ Anydesk                                      │ │
│ │ 1218740083/antraium/администратор           │ │
│ │ ...                                          │ │
│ │ ↕️ (20 строк)                                │ │
│ └──────────────────────────────────────────────┘ │
│                                                  │
│ [💾 Сохранить]  [❌ Отмена]                     │
└──────────────────────────────────────────────────┘
```

**Поля**:
1. **Название организации** (текстовое поле)
   - Одна строка
   - Обязательное
   - Валидация: не пустое

2. **Содержимое** (textarea)
   - Многострочное
   - Моноширинный шрифт
   - Растягивается по вертикали
   - 20 строк по умолчанию

**Кнопки**:
- "💾 Сохранить" (зеленая) - сохраняет изменения
- "❌ Отмена" (серая) - возвращает к режиму просмотра

**Горячие клавиши**:
- `Ctrl+S` - сохранить

## 7. Модальное окно "Новая организация"

Открывается по кнопке "+ Новая wiki-страница"

```
        ┌──────────────────────────────┐
        │ Новая организация        [×] │
        ├──────────────────────────────┤
        │                              │
        │ Название организации:        │
        │ ┌──────────────────────────┐ │
        │ │ _                        │ │
        │ └──────────────────────────┘ │
        │                              │
        │ Содержимое (опционально):    │
        │ ┌──────────────────────────┐ │
        │ │                          │ │
        │ │                          │ │
        │ │                          │ │
        │ │                          │ │
        │ └──────────────────────────┘ │
        │                              │
        ├──────────────────────────────┤
        │      [Создать]  [Отмена]     │
        └──────────────────────────────┘
```

**Особенности**:
- Затемненный фон (backdrop)
- Центрирование по середине экрана
- Кнопка "×" для закрытия
- Enter в поле названия = создать

**Валидация**:
- Название обязательно
- Содержимое опционально
- Если название пустое → alert

**После создания**:
- Автоматический переход на созданную страницу
- Новая организация появляется в левом меню

---

## 8. Dropdown меню "Экспортировать"

```
┌──────────────────┐
│ ⬇️ Экспортировать│
│       ▼          │
│   ┌──────────┐   │
│   │ 📄 PDF   │   │
│   │ 📝 TXT   │   │
│   └──────────┘   │
└──────────────────┘
```

**Поведение**:
- Клик → показать меню
- Клик вне меню → скрыть
- Клик на формат → скачать файл

**Форматы**:
- **PDF**: Красивый документ с заголовком и форматированием
- **TXT**: Чистый текст без форматирования

**Имя файла**:
- `Название_организации.pdf`
- `Название_организации.txt`

## 9. Диалог подтверждения удаления

```
┌─────────────────────────────────────────┐
│ ⚠️  Подтверждение                       │
├─────────────────────────────────────────┤
│                                         │
│ Вы уверены, что хотите удалить          │
│ организацию "ALPHUTA"?                  │
│                                         │
│         [OK]        [Отмена]            │
└─────────────────────────────────────────┘
```

**Встроенный**: Используется стандартный `confirm()` браузера

**После подтверждения**:
- Организация удаляется из БД
- Автоматическое перенаправление на главную страницу
- Организация исчезает из левого меню

## 17. Особенности реализации

### JavaScript:

- Vanilla JS (без jQuery)
- Fetch API для AJAX
- Event delegation
- Modal без библиотек

### CSS:

- Flexbox для layout
- Custom scrollbars (webkit)
- Transitions для плавности
- БЕЗ препроцессоров (чистый CSS)

### Доступность:

⚠️ **Ограниченная** - проект для внутреннего использования.

Что НЕ реализовано:
- ARIA атрибуты
- Keyboard navigation
- Screen reader support
